12. Верно ли определение оператора «пересечение»: «Для двух реляционных таблиц R1 и R2 одинаковой арности и с совпадающими типами полей формируется РТ R с записями, входящими в одну из РТ»?

 

13. Верно ли определение оператора «проектирование»: «Из реляционной таблицы R1 оператор выбирает заданные типы полей и размещает в указанном порядке в таблице R»?

 

14. Верно ли определение оператора «выбор»: «Из реляционной таблицы  R1 по заданному критерию выбирается требуемая совокупность значений»?

 

15. Верно ли определение оператора «соединение»: «На основе двух реляционных таблиц R1 и R2 имеющих одно или несколько полей идентичных типов, формируется реляционная таблица R, состоящая из записей, каждая из которых является соединением в одну тех записей таблиц R1 и R2, у которых совпадают значения полей всех идентичных типов»?

 

16. Верно ли определение оператора «умножение»: «Для двух реляционных таблиц R1 и R2 соответственно арности K1 и K2 формируется результирующая реляционная таблица R арности (K1 * K2), записи которой представляют собой конкатенацию каждой записи из таблицы R1 с каждой записью из таблицы R2»?

 

17. Задача оптимизации при реализации запросов заключается в:
